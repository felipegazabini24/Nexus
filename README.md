# Nexus — Marketing & Gestión de Anuncios

Sitio web y base de la agencia **Nexus**: gestión de anuncios (Meta, Google, TikTok), diseño web premium y estrategia de performance para clientes.

## Estructura
```
Nexus/
├─ index.html      # Landing page principal
├─ css/styles.css  # Estilos (tema oscuro premium, acentos dorados)
├─ js/main.js      # Animaciones: reveal on scroll, contadores, nav
└─ assets/         # Imágenes, logos, recursos
```

## Páginas del sitio
